\# MLA Formatter



\## Description

An automated browser skill that applies strict MLA formatting to a Google Document. It selects all text and utilizes Google Docs' internal search menus to apply Times New Roman font, 12pt size, and double spacing without requiring manual CSS selector targeting.



\## Parameters

\- `target\_url` (string): The full URL of the Google Doc to be formatted.



\## Use Case

Trigger this skill as the final action \*after\* a multi-session drafting process is complete to provide a polished, human-like revision pass on the document.
